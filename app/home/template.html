curvature

curvature is a lightweight javascript framework with an emphasis on straightforwardness.

Author's note: Although this is an extremely fun project to work on, I use shop-standard tools when working in the office.

npm

the recommended way to get curvature is via npm:

$ npm i curvature

cdn

curvature is also available from unpkg:

minified:

<script src = "https://unpkg.com/curvature@[[version]]/dist/curvature.min.js"></script>

uncompressed:

<script src = "https://unpkg.com/curvature@[[version]]/dist/curvature.js"></script>

Sonic 3000

Sonic 3000 is a browser based 2d platforming game styled aftet the Sonic games for Genesis from the 1990s.

Features full gamepad support, P2P multiplayer, eight playable characters, elemental sheilds, new moves, and vehicles for both land and air.

View the README for full control instructions.

view the live demo

Nynex 95

Nynex 95 simulates the look and feel of Windows 95 in the browser. Features include task management, a basic terminal, "pop-out" windows with a working task bar (double click the titlebars to "pop" windows out of the browser).

The project also features a text editor inside the "Repo Browser" app that can edit the code in its own repo live when authenticaed via the github api.

view the project on github

view the live demo

Bit Canvas

Bit Canvas parses Pokemon Red/Blue ROM files and allows the user to extract images and monster statistice directly from the gameboy machine code.

view the project on github

view the live demo

php-wasm

php-wasm allows the user to write and execute PHP live in the browser, fully client-side. The system is powerful enough to boot Drupal 7 and display its homepage, with extensions like SQLite modified to compile statically. Cookie-based logins are still yet-to-be implemented.

view the project on github

view the live demo

Reckon

Reckon is a parsing library inspired by Perl 6 Grammars.

Demo features parsing & rendering of ANSI color & formating codes.

Supports 4, 8 and 24 bit coloring, text & binary input, as well as the terminal BEL (type "\\a" in the input field).

view the project on github
