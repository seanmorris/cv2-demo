Forms Demo

Forms can be generated by passing an object describing the desired fields & properties. Keys prefixed with "_" will affect the form itself rather than generating fields. Other keys will map to fields.

  • _method - The "method" attribute of the form element.
  • _action - The "action" attribute of the form element.

Fields are object with 0 or more of the following keys

  • name - Field name attribute (key will be used if not provided.)
  • title - Field title (key will be used if not provided.)
  • type - Field Type
  • attrs - Additional attributes for field tag (k/v object)

Note form submisions are disabled in the editor

[[basic]]

Field Groupings

Fields may be grouped with the fieldset field type.

Fieldset definitions may have the following keys in additions to the standard keys:

  • children - Array or object listing child field.
  • array - Boolean determining if field names should cascade.

Array field names will appear normal in output JSON but will have their name attributs in the form fieldset[fieldname]. Nested fieldsets can cascade this property to produce names like fieldset[subset][fieldname].
