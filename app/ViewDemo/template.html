Binding Values

Scalars

Scalar can be bound with double square brackets:

[​[time]]

[[time]]

Two way binding is also supported:

<button cv-on = "click:clear('twoWay')">X</button>
<input cv-bind = "twoWay">: [​[twoWay]]
:[[twoWay]]

HTML is automatically escaped unless "$" is used:

[​[time|addItalicTags]]

[[time|addItalicTags]]

[​[$time|addItalicTags]]

[[$time|addItalicTags]]

Arrays & Objects

Arrays and objects may be looped over with the cv-each attribute:

<ul cv-each = "list:item:i"/><li>[​[item]]</li></ul>
  • [[item]]

Methods

Events

Methods can be bound to events:

<button cv-on = "click:clear('twoWay')">X</button>
:[[twoWay]]

Transformations

Values can also be transformed with methods:

[​[time|reverseString]]

[[time|reverseString]]

Values of any type may be transformed by methods:

[​[list|toJson]]

[[list|toJson]]

Templates

Reusable templates can be created with the cv-template attribute:

<template cv-template = "item">
	<li><input type = "checkbox">[​[item]]</li>
</template>

Templates may be reused within the same view with the cv-slot attribute:

<ul cv-each = "list:item:i" cv-slot = "item"/></ul>
<ol cv-each = "list:item:i" cv-slot = "item"/></ol>
