config

Configuration may be set in a few ways:

  • Calling Config.set('name', 'value') on curvature/base/Config.
  • Calling Config.init({name: value, ...}) on curvature/base/Config.
  • Creating a top-level Config class and setting static properties on it.

Configs may be read in two ways:

  • Calling Config.get('name') on curvature/base/Config.
  • Reading a static property from a top level Config class.
