bags

A Bag object is a container for other objects that is similar in certain ways to a Set or a Map. It has the following properties:

  • It can only hold other objects, no scalars.
  • It can hold only one reference to each object.
  • It will maintain the order that objects were added in.
  • It will automatically assign an ID to each object.
  • It can be passed to a View to be iterated during rendering.
  • It is an EventTarget and can be watched for additions and removals.
  • The add/remove events can be canceled.
  • The add/remove events can be canceled.
