mixin

Mixins provide a multiple-inheritance like facility.

The Mixin class provides 3 static methods to create mixins:

  • Mixin.with(...mixins)
  • Mixin.from(baseClass, ...mixins)
  • Mixin.to(toClass, ...mixins)

Using Mixins:

Mixin.with(...mixins)

The simplest method, Mixin.with() takes a list of mixins and applies them to an emply base class. It then returns the resulting class, which can then be extended or instantiated.

Mixin.from(baseClass, ...mixins)

Mixin.from() takes a base class and a list of mixins. It extends the base class, and applies the mixin to the new class before returning it.

Mixin.to(toClass, ...mixins)

Mixin.to() takes a class and a list of mixins. It then applies the mixins to the class, in-place.
