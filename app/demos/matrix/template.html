Install matrix-api with the npm package manager.

$ npm install matrix-api

Then, import or require the module in your javascript:

import { Matrix } from 'matrix-api/Matrix';
or 
const Matrix = require('matrix-api/Matrix').Matrix;

You can also include Curvature & Matrix-Api API libraries from a CDN with:

<script src = "https://unpkg.com/curvature@0.0.66-p/dist/curvature.min.js"></script>
<script src = "https://unpkg.com/matrix-api@0.0.0-d/dist/matrix-api.min.js"></script>

Or pull a version with -just- enough Curvature to run Matrix API:

<script src = "https://unpkg.com/matrix-api@0.0.0-d/dist/matrix-api.standalone.js"></script>
