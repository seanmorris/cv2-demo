curvature

curvature is a lightweight javascript framework with an emphasis on straightforwardness.

npm

the recommended way to get curvature is via npm:

$ npm i curvature

cdn

curvature is also available on unpkgr:

<script src = "https://unpkg.com/curvature@0.0.65/dist/curvature.js"></script>

source

the curvature source code is available on github:

$ git clone https://github.com/seanmorris/curvature-2
curvature source

demo source

the demo source code is also on github:

$ git clone https://github.com/seanmorris/cv2-demo
demo source
