Experimental Scrollers

These scroll boxes automatically remove any DOM elements that are no longer visible to the user. This allows us to make large scrollable boxes without wasting large amounts of computing power on DOM calculations for records that may never become visible.

Simple Scroller

This scroller uses an array of 1,000,000 integers.

Arrays and Record Sources are interchangeable here.

[[arrayScroller]]

String Scroller

This scroller uses a Record Source that returns strings.

An array of strings could easily have been used instead of a Record Source.

[[stringScroller]]

View Scroller

This scroller uses a Record Source that returns curvature View objects.

There is a 750ms simulated load time when the view is first accessed.

[[viewScroller]]

Grid Scroller

This scroller uses CSS grid, and a custom template to simulate a table. Its RecordSource returns plain objects.

The number of records and row height can be altered on the fly as well, this is common to all Scrollers.

Counting both the texboxes & sliders, there are 2,000,002 input fields avaiable below, each mapped to an object in memory.

 
[[gridScroller]]
